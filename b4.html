    TEXT TRANSFORMATION

  • The text transformation classes in bootstrap are used to apply different styles to text,such as converting it to uppercase,lowercase the first letter of each word.These classes are bootstrap utility classes.

  • TEXT-LOWERCASE :- This class converts oll the letters in the text to lowercase.

  • Example:

    • HELLO WORLD!

  • TEXT-UPPERCASE :- This class converts all the letters in the text to uppercase.

  • Example:

    • hello world!

  • TEXT-CAPITALIZE :- This class capitalize the first letter of each word in the text.

  • Example:

    • hello,world!
